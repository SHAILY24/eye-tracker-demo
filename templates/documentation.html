About the Project

The Eye Tracker Demo is an interactive web application that utilizes WebGazer.js to track user gaze in real-time, enabling hands-free interaction with on-screen elements through eye movements alone.

Inspiration

This project explores the potential of eye-tracking technology for creating more accessible and intuitive human-computer interfaces, demonstrating how gaze-based interaction can enhance user experiences and provide alternative input methods for users with mobility challenges.

How to Use

  1. Ensure your webcam is enabled and accessible by your browser.
  2. Click on "Start Game" on the home page.
  3. Allow the application to calibrate your eye movements (this may take a few moments).
  4. Once calibration is complete, use your gaze to find and focus on the target to interact with it.
  5. If you're on a mobile device, please ensure your device supports webcam access and rotate it to landscape mode for the best experience.

Troubleshooting

If the eye-tracking does not seem accurate, try the following:

  • Make sure your face is well-lit and clearly visible to the webcam.
  • Avoid background distractions and strong light sources behind you.
  • Try recalibrating by refreshing the page and starting the game again.
  • Ensure you're using a compatible browser (latest versions of Chrome or Firefox recommended).
